1. 주제 : 책 주문 시스템
1-1 시나리오 
- 고객이 책을 선택하여서 주문한다.
- 주문이 들어오면 상품을 배송한다.
- 배송이 시작되면 재고량를 감소시킨다. 
- 고객은 주문을 취소할 수 있다.
- 주문이 취소면 배달이 취소된다.
- 배달이 취소되면 재고량을 증가시킨다.
- 고객은 언제든지 주문한 내역과 배송 상태를 조회한다.
